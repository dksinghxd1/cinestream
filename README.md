# CineStream

A full-stack movie streaming app inspired by Netflix, built with React (frontend), Node.js/Express (backend), and TMDB API integration. Includes a GitHub Actions CI/CD pipeline for build, test, and deployment.

## Structure
- `client/` — React frontend
- `server/` — Node.js/Express backend

## Features
- Browse and search movies using TMDB API
- Responsive, modern UI
- Ready for CI/CD with GitHub Actions

## Getting Started

### Prerequisites
- Node.js (18+ recommended)
- TMDB API Key ([get one here](https://www.themoviedb.org/))

### Setup

1. Clone the repo
2. Install dependencies for both `client` and `server`
3. Add your TMDB API key to the environment files
4. Run both apps locally

### CI/CD
- GitHub Actions workflow included for build, test, and deploy (see `.github/workflows/ci.yml`)
